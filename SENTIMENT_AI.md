## ⚡ Quick Start

**Command Prompt (CMD):**
```cmd

# 2. Migrate database
php artisan migrate

# 3. Start API
start_sentiment_api.bat

# 4. Read docs
cd python_ai_service
start INDEX.md
```

